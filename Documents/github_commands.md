#  Git commands:

### Initializing a Repository:
>
> This command initializes a new Git repository
>
>> `git init`

### Cloning a Repository:
>
>This command creates a local copy of an existing remote repository from the given URL.
>> `git clone https://github.com/user/repo.git`

### Staging Changes:
>
>This command adds index.html to the staging area, marking it for inclusion in the next commit.
>>`git add index.html`
>
> This command adds index.html to the staging area, marking it for inclusion in the next commit.
>>`git add .`
